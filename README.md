# Grid overlay
Накладывает сетку на сайт для тестирования верстки.

Grid overlay это расширения для google chrome.


## Преимущества
* Доступное (a11y)
* Семантичное
* Использует нативные контролы
* Пресеты популярных сеток
  * 960grid
  * 8 point grid
* Вертикальный ритм


## Использование
1. Установить из магазина Google Chrome
2. Нажать на иконку
3. Пользоваться
